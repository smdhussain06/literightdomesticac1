# 🔦 LiteRight Academy — Lighting Design Masterclass

Welcome to the **Lighting Design for Interior Spaces** complete course repository! 

This repository hosts a beautifully structured, comprehensive HTML course package tailored for the **LiteRight Academy LMS**. It contains everything a student or client needs to understand the art and science of lighting design.

## 🚀 Live Demo & Course Access
**[View the Live Course Here](https://smdhussain06.github.io/literightdomesticac1/)**

## 📚 What's Inside?
This single-file offline-compatible web application contains:
- **Comprehensive Course Structure:** 6 modules starting from the basics of light and perception, advancing through spatial analysis, concept drafting, and technical representation.
- **Synthesia Video Scripts:** 14 detailed, timecoded, scene-by-scene video scripts meticulously formulated for AI digital avatars.
- **Interactive Trees & Assets:** Instantly navigable UI tracking video runtimes, supplementary PDFs, and checklist modules.
- **Cross-device Readability:** A neat, incredibly readable UI built for laptops, tablets, and mobile phones alike.
- **Print-What-You-See Ready:** A built-in PDF styling script that cleanly formats your specific active lesson into a beautiful, textbook-quality document when you hit File -> Print.

## 🛠️ Usage
This repository is pre-configured to be hosted on **GitHub Pages**. Simply access the domain to read the content. It operates entirely on Vanilla CSS and semantic HTML so there are no dependencies to install.
