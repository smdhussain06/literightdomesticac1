🎓 Final Project: Complete Lighting Design

Capstone project demonstrating all skills from the course

Part 1: Conceptualization (30%)

  • Space & User Audit
  • Material Catalog
  • Natural Light Analysis
  • Core Idea statement (1 sentence)
  • Set of Rules (5–8 rules)
  • Mood Board (8–12 reference images)
  • Richard Kelly concept identification
  • Layer and hierarchy plan (min 3 layers)

Part 2: Technique (30%)

  • Fixture selection for each layer (full specs)
  • Color temperature plan
  • Justification for each fixture (linked to concept)

Part 3: Documentation (40%)

  • Floor plan with fixture locations and symbols
  • 2+ sections/elevations with beam distributions
  • 1+ perspective drawing with color-coded light
  • Cross-section details for 2+ fixtures
  • Assembled conceptual presentation

Evaluation Criteria
